#### 9.7.19.1. Miscellaneous Instructions:brkpt


`brkpt`

Breakpoint.

Syntax

brkpt;

Description

Suspends execution.

PTX ISA Notes

Introduced in PTX ISA version 1.0.

Target ISA Notes

`brkpt` requires `sm_11` or higher.

Examples

brkpt;
    @p  brkpt;
